# AoCC

Advent of Code Challenges


## How to check

Choose the year and give a day number like `day1` as followings.

### Rust

```zsh
% cd src/2023
% cargo run -p aocc-2023 day1 --release
```

### Q

```zsh
% cd src/2023
% q ./src/q/day1.q
```
